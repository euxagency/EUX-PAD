Pickup and delivery date selection for WooCommerce, with admin rules, settings, and checkout integration.

== Description ==

EUX Pickup & Delivery adds a dedicated Pickup & Delivery step to your WooCommerce checkout flow. Customers can choose pickup or delivery, select a date and time where applicable, and continue to checkout with their choice stored on the order.

**Requirements**

* WordPress 5.0 or higher
* WooCommerce 7.0 or higher
* PHP 7.4 or higher

Features include configurable global and per-mode settings, delivery suburbs, scheduling rules, and compatibility with WooCommerce custom order tables (HPOS).

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or upload the ZIP via **Plugins → Add New → Upload Plugin**.
2. Activate **EUX Pickup & Delivery** through the **Plugins** screen.
3. Ensure **WooCommerce** is installed and active.
4. Configure settings under the Pickup & Delivery admin menu.

== Frequently Asked Questions ==

= Does this plugin work without WooCommerce? =

No. WooCommerce must be installed and active.

= Where are the settings? =

After activation, use the Pickup & Delivery entries in the WordPress admin menu (Global, Pickup, Delivery, Rules).
